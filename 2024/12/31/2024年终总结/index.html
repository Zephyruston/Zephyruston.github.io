2024年终总结

Created2024-12-31|Updated2024-12-31
|Post Views:

现在是 24 年最后一天，评价一下自己这一年吧。今晚是跨年夜，我独自享受着图书馆安静的氛围，码下这篇文章。

先谈学习上吧。年初到大二下学期了，我在对计算机专业的就业及学习路线之类的都是迷茫的，但是我遇到了个好老师(孙孔峰老师)，他教授数据库概论这门课，从他课上我学会了去写 sql 语句(对于当时迷茫，浑浑噩噩混日子，code 水平差的一批的我来说算是增加了点信心)，在课上他还跟我们分享就业方向。在课后，自己便去认真了解计算机就业相关的信息。也是重新拾起了半吊水平的 c++，学着去刷力扣，去看黑马的课，自己尝试着手搓 avl tree，实现一些数据结构等。在暑假无意中翻到了一个网站，北大大佬搞的cs 自学指南，在里面发现了很多国外名校的公开课，我如同发现新世界，挑选了里面的 cmu15445 跟着 andy 学数据库底层的知识，也是让我受益匪浅。当初看这个课是因为自己对数据库有点了解，然后课程实验也是用 c++去写，便开始了学习，不过课程的实验还没有做完 😫(自己太菜了)。到大三上学期，有一门课数据库综合课程设计，要求要搞可视化啥的，一开始想着用 qt 搞？嗯，算了吧，还是搞网页吧。然后又因为以后想看 MIT6.824，这个课的实验是 go 去写，然后想让自己早点学习 go，就在这个课设搞了 goweb。搞完又是一发不可收拾，发现自己对 c++没有这么热爱，还写出一坨内存泄漏的代码(miniob 比赛)，了解了很多 web 开发的东西，云原生，微服务之类的词，我这才意识到这个可能是我感兴趣的方向，以后就业争取往这里看齐。又在偶然间听到 rust 的名号，别人一顿吹，关键在开源社区也挺火的，我也就跟风学了一下(未来仍打算认真深入学习)，用于 stm32 开发中，完成了智能设备接口课程的实验。再谈谈英语，自己一年两考六级，听力还是差的一批，感觉过级无望 😭😭。

25 年学习展望：1. 坚持刷算法题。2. 坚持英语学习。3. 争取多看书。4. rust 学习，争取实践到项目中。5. 做好秋招准备。

生活上，今年跑了两场马拉松，一场清远半马，一场广州全马。广马的戒断反应真的很大，明年还想去跑一趟。希望自己坚持跑下去，也远离伤病，早日把半马 145，全马 400 的目标实现。

总结：24 年找到了自己真正想干的事，也有热爱的事，也许还有遗憾，没能在今年的半马赛事 sub145，still a single dog。25 年要面对就业压力了，更多挑战，向业内大佬学习，拿到满意的 offer。
